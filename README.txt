◆作品説明
冬だ！雪だ！真っ白だ！
世界が白く染まる前に自分で真っ白に染めちゃおう！

◆工夫した点
・テクスチャを縛りで色を表現するためマテリアルカラーを変更しました
・白く染めるのを邪魔するNPCを実装し、ソロプレイでも張り合いが生まれるようにしました。

◆操作方法

「キーボード / マウス」
・WASDキー			→	移動
・マウス操作		→	カメラ操作
・スペースキー		→	雪かけ(ブレーキ) / 決定

「XInput対応ゲームパッド」
・左スティック		→	移動
・右スティック		→	カメラ回転
・Rボタン, Lボタン	→	雪かけ(ブレーキ) / 決定
